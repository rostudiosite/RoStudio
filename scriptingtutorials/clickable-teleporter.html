How to make a clickable teleporter in Roblox Studio?

Today we will be creating a clickable teleporter in Roblox Lua. Throughout this tutorial, you will be learning how to use click detecters.

Setup:

Inside Workspace create a part. You can name it whatever you want. Inside the part add a click detecter and a script. Name the click detected "ClickDetecter".

It should look something like this:

Scripting:

Now we will move on to scripting.

First we will start with our variables.

The "Part" variable will be for our click detecter part.

The "Detecter" variable will be for our click detecter.

                    
local Part = script.Parent
local Detecter = script.Parent.ClickDetecter

Now we will get whenever someone clicks the part using the "MouseClick" event on with click detecter.

                    
local Part = script.Parent
local Detecter = script.Parent.ClickDetecter

ClickDetecter.MouseClick:Connect(function(PlayerWhoClicked)

end)

Finally, we will teleport the player using the function "MoveTo" on the character model. The position must be a Vector3.

                    
local Part = script.Parent
local Detecter = script.Parent.ClickDetecter

ClickDetecter.MouseClick:Connect(function(PlayerWhoClicked)
    PlayerWhoClicked.Character:MoveTo(Vector3.new(0, 2, 0))
end)

You can also teleport to a parts position with PartVariable.Position.

We are done!
