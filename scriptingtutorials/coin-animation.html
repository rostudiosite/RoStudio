How to make a coin animation in Roblox Studio?

Today we will be creating a coin animation in Roblox Lua. Throughout this tutorial, you will be learning how to use Vector3.

Setup:

Make a part inside of the workspace and anchor it. Once you have done that, make a script inside of the part.

It should look like this:

Scripting:

Now we will move on to scripting.

Just folow my easy steps.

1. We get the variables that we need for this which is only the part.

                    
local Part = script.Parent

2. We create a loop which repeats over and over again.

                    
local Part = script.Parent

while true do wait()

end

3. Final Step, actually making it rotate. Depending on the number or way you rotate it, that's your choice but im making it rotate on the sides like a coin. Nice and easy.

                    
local Part = script.Parent

while true do wait()
    Part.Orientation += Vector3.new(0,15,0)
end

(Disclaimer: Vector3 is used to either change the size,rotation, and more of a part for example in the workspace or any object. Just not UI)

Now that should be it! Let's test! Perfect!
