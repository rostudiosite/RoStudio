How to make a shift to sprint in Roblox Studio?

Today we will be creating a shift to sprint in Roblox Lua. Throughout this tutorial, you will be learning how to use UserInputService.

Setup:

Inside PlayerScripts > StarterCharacterScripts create a Local Script. You can name it whatever you want.

Scripting:

Now we will move on to scripting.

For our variables we will need two services. Players and UserInputService.

We will also have variables for our player and character.

                    
local Players = game:GetService("Players")
local UserInputService = game:GetService("UserInputService")

local Player = Players.LocalPlayer
local Character = Player.Character

Now let's move on to the user input part. We will be using Players and UserInputService to get when the player presses Shift or let's go.

We can check when the player presses the key with InputBegan and when they let go we can check with InputEnded.

                    
local Players = game:GetService("Players")
local UserInputService = game:GetService("UserInputService")

local Player = Players.LocalPlayer
local Character = Player.Character

UserInputService.InputBegan:Connect(function(input)

end)

UserInputService.InputEnded:Connect(function(input)

end)

Now let's check if the input key code is Shift. If so, we will make their walk speed higher.

We will also be creating two more variables that contain the default walk speed and the sprinting walk speed.

                    
local Players = game:GetService("Players")
local UserInputService = game:GetService("UserInputService")

local Player = Players.LocalPlayer
local Character = Player.Character

local DefualtSpeed = 16
local SprintingSpeed = 25

UserInputService.InputBegan:Connect(function(input)
    if input.KeyCode == Enum.KeyCode.LeftShift then
        Character.Humanoid.WalkSpeed = SprintingSpeed
    end
end)

UserInputService.InputEnded:Connect(function(input)

end)

Finally, we will detect when the player has released the shift key and set their walk speed to normal.

                    
local Players = game:GetService("Players")
local UserInputService = game:GetService("UserInputService")

local Player = Players.LocalPlayer
local Character = Player.Character

local DefualtSpeed = 16
local SprintingSpeed = 25

UserInputService.InputBegan:Connect(function(input)
    if input.KeyCode == Enum.KeyCode.LeftShift then
        Character.Humanoid.WalkSpeed = SprintingSpeed
    end
end)

UserInputService.InputEnded:Connect(function(input)
    if input.KeyCode == Enum.KeyCode.LeftShift then
        Character.Humanoid.WalkSpeed = DefualtSpeed
    end
end)

We are done!
